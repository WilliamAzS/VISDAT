# YouTube Pulse — Dashboard Visualisasi Data

Dashboard ini dibuat untuk Proyek Akhir CA4XB3 Visualisasi Data. Kami menggunakan data channel YouTube global dari Kaggle, lalu menampilkannya ke dalam website interaktif yang bisa dibuka langsung melalui Vercel.

## Dataset

Dataset: Kaggle — Global YouTube Statistics 2023  
URL: https://www.kaggle.com/datasets/nelgiriyewithana/global-youtube-statistics-2023

## Fitur

- 5 chart interaktif dengan Chart.js
- Tooltip, animasi, filter negara, filter kategori, search, sort, dan slider Top-N
- Quick preset untuk melihat Music, Education, India, United States, dan upload terbanyak
- Channel Spotlight yang berubah saat user klik chart atau baris tabel
- KPI count-up dan insight otomatis
- Tabel data dinamis dengan highlight channel terpilih
- Download CSV hasil filter
- Light/dark mode
- Desain sleek dan modern

## Deploy ke Vercel

Website ini adalah static site. Import repository ke Vercel, pilih preset `Other`, lalu klik Deploy.
